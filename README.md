# SONGNOW_framework
Web application for playing songs. Developed using php framework.

## Prerequisites
* Apache
* Mysql

## Developed with
* PHP Framework
* JS
* JQuery
* Mysql / Database

## API used
* Google maps
* Firebase
* Mailgun
